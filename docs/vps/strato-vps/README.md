# STRATO VPS Reference Flake

This directory contains the older standalone STRATO VPS reference flake that
predated the integrated `albaldah` machine in the main repo.

## Structure

- reference flake entrypoint
- diagnostics snapshot from the source system
- older host-local machine files for the standalone layout

## Role

Keep this directory as historical reference material. It helps explain where
the integrated VPS work came from, but it is no longer the main path for the
repo-managed host definition.
