# Dotfiles Docs

`dotfiles/docs/` is a local-area documentation pocket for the desktop and
shell config under `dotfiles/`. It is not a second repo-wide documentation
system.

## Sections

- `setup/`
  - current workflow and behavioral guidance for the live desktop setup
- `decisions/`
  - local architecture or workflow decisions that are still worth keeping
- `recaps/`
  - historical notes and dated context

## Placement Rules

- Keep durable local workflow guidance here when it is specific to `dotfiles/`.
- Keep historical recaps here only when they still help explain local context.
- Do not keep finished migration plans, temporary task lists, or one-off AI
  bootstrap prompts here once the work is complete.
